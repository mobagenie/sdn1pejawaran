Hero Terbaru Mobile Legends 2019

Published : May 7, 2019, 20:15 Reading time : 2 m and 46 s.

MobaGenie - Berikut MobaGenie update 6 Hero baru di Mobile Legends yang dirilis Moonton Tahun 2019 Lengkap Dengan Gambar dan beberapa info penting yang mobagenie rangkum dari Gcube. Beberapa hero di postingan ini ada yang sudah rilis, ada juga yang baru bisa digunakan di Advanced server.

Moga postingan kali ini bisa membantu buat kalian yang penasaran dengan hero terbaru di Game Mobile Legends.

1. X.Borg

X.Borg


Hero paling baru dan menjadi hero terahir yang dirilis Moonton untuk Mobile Legends sampai postingan ini ditulis adalah X.Borg. berikut ini info detail dari hero X.Borg

Release Date : 11/04/2019 X.Borg rilis di Advanced Server
Role : Fighter

Kelebihan
- Mempunyai Skill dengan AoE dan Burst Damage yang besar
- Memiliki daya tahan yang kuat untuk seorang Fighter
- Late Game akan membuat X.Borg mendominasi
- Regenerasi terus menerus yang membuat X.Borg susah dibunuh
Kekurangan
- Sulit untuk melakukan lock target ke arah lawan
- Skill Ultimate bisa menjadi bahaya apabila salah penggunaanya.
- Kurang memiliki efek Crowd Control sebagai seorang Fighter.

2. Granger

Granger

di posisi kedua hero terbaru adalah hero Granger

Release Date : 22/03/2019 Granger rilis di Advanced Server
Role : Marksman

Kelebihan
- Mempunyai Burst Damage yang besar
- Dapat mengincar lawan dari kejauhan
- Late Game akan membuat Granger mendominasi
Kekurangan
- Sangat mudah diincar oleh Assasin lawan
- Penggunaan Skill 1 merepotkan dan merupakan sumber damage utama
- Ketika peluru habis, basic attack tidak dapat digunakan sesuka kita
- Sangat susah dimainkan saat early game

Trailer :

3. Esmeralda

Esmeralda

Release Date :

Advanced Server (28/02/2019)
- Esmeralda rilis di Advanced Server

Public Server (03/04/2019)
- Esmeralda rilis di Public Server

Patch 1.3.68 (16/04/2019)
- Stardust Dance: Menyesuaikan cooldown dari 7/6,3/5,6/4,9/4,2/3.5 menjadi 5.5/5,1/4,7/4,3/3,9/3.5.
- Mengoptimalkan ikon skill

Role : Mage/Tank

Kelebihan
- Sebagai Mage Esmeralda memiliki Darah yang cukup tebal
- Memiliki 2 type damage yang membuat lawan bingung melindungi diri
- Damage yang dimiliki cukup kuat untuk melakukan 1 vs 1
Kekurangan
- Kurangnya Crowd Control jika disetting sebagai Tank
- Untuk mengejar target cukup sulit
- Skill yang dimiliki kurang memiliki Damage AoE

Trailer :

4. Khufra

Khufra

Release Date :

Advanced Server (04/01/2019)

Khufra rilis di Advanced Server
Public Server (12/03/2019)

Khufra rilis di Public Server

Role : Tank

Kelebihan
- Sangat tebal dengan physical dan magic defense
- Sangat berguna untuk memulai ganking atau war
- Memiliki skill dengan Crowd Control yang banyak
Kekurangan
- Hampir semua Skill yang dimiliki memiliki Cooldown yang cukup lama
- Skill yang dimiliki tidak pasti (timing harus akurat)
- Skill yang dimiliki mempunyai damage yang kecil

Trailer :

5. Guinevere

Guinevere

Release Date :

Advanced Server (17/01/2019)

Guinevere rilis di Advanced Server
PublicS Server (21/02/2019)

Guinevere rilis di Public Server

Role : Fighter / Mage

Kelebihan
- Sebagai Mage Guinevere memiliki Darah yang cukup tebal
- Guinever bisa menjadi inisiator yang baik untuk memulai ganking atau war
- Memiliki skill dengan Crowd Control yang cukup lama
- Memiliki Mobilitas yang tinggi
Kekurangan
- Salah menggunakan combo akan berakibat fatal bagi Guinevere
- Skill yang dimiliki tidak pasti (timing harus akurat)
- Control yang cukup sulit membuat bingung

Trailer :

6. Vale

Vale

Release Date :

Patch 1.2.84 (8/6/2018)

- Pertama kali rilis di Advanced Server

Public Server (29/01/2019)

- Vale rilis di Public Server

Role : Mage

Kelebihan
- Burst damage yang sangat tinggi
- Seluruh skillnya dapat diupgrade sesuai dengan selera
- Bersinergi dengan sangat baik bersama hero yang mampu menarik lawan ke satu tempat
Kekurangan
- Tidak memiliki skill untuk melarikan diri
- Sangat rentan dengan efek crowd control
