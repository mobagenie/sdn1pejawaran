Build Hayabusa Offlaner Versi Baim Wong

Published : Mar 20, 2021, 11:19 Reading time : 0 m and 40 s.
Pict via YT / Ikhsan Lemon


MobaGenie,- Beberapa hari lalu Ikhsan Lemon Bermain Mobile Legends melawan Baim Wong dalam Event "Berpetualang Cari Legend Bareng Baim Wong" yang disiarkan langsung melalui Livestream Youtube pribadi Ikhsan Lemon. Di event tersebut, Baim memilih Hayabusa namun bukan sebagai hero Carry atau Jungler melainkan Sidelaner atau Offlaner. 

Untuk Emblem, Baim memilih Emblem Assasin dengan Talent Killing Spree. sedang Spell, Baim memilih Execute.  


Pict via YT / Ikhsan Lemon

untuk build, item pertama yang dibeli yaitu Warrior Boot, kemudian item kedua yaitu Endles Battle, dilanjut ke item ke tiga yaitu BoD. sayangnya untuk 3 item terahir belum terbeli sampai pertandingan berahir. namun jikalau melihat dari 3 item terahir, sepertinya Baim mau membeli BoD lagi dilanjut item Blodlust Axe?

Pict via YT / Ikhsan Lemon

Pertandingan pun berahir dengan Skor 29:9 dimana Kemenangan diraih oleh Team Ikhsan Lemon.
